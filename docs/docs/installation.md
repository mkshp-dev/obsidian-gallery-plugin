# Installation

## From the community store (recommended)

1. Open Obsidian **Settings**.
2. Go to **Community plugins** and disable **Restricted mode**.
3. Click **Browse** and search for **Image Gallery**.
4. Click **Install**, then **Enable**.

## Manual installation

1. Download `main.js`, `manifest.json`, and `styles.css` from the [latest release](https://github.com/mkshp-dev/obsidian-gallery-plugin/releases/latest).
2. Create the folder `<vault>/.obsidian/plugins/image-gallery/`.
3. Copy the three files into that folder.
4. Open Obsidian **Settings → Community plugins** and enable **Image Gallery**.

## Requirements

- Obsidian 0.15.0 or later
- Works on desktop and mobile
